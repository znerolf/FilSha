# 🔐 Filsha File Sharing with End-to-End Encryption

Filsha is a secure file-sharing web application that uses End-to-End Encryption (E2EE) to protect files and user data during upload, transfer, and download.

---

## ✨ Features

- 🔒 End-to-End Encryption
- 📂 Secure File Upload & Download
- 👤 User Authentication System
- 🔑 Password Protected Sharing
- ⚡ Fast and Lightweight
- 🛡️ Secure File Transfers
